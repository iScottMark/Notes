**Python 字符串前各字母的含义**

- u: 以 unicode 进行编码，主要针对中文字符
- r: 去除反斜杠 `\` 的转义机制
- b: 表示后面字符串是 bytes 类型，主要针对 web 编程。
- f: 表示字符串内支持大括号内的 Python 表达式
  - `print(f'{name} done in {time.time() - t0:.2f} s') `
  - `processing done in 1.00 s`
